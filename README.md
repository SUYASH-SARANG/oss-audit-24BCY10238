## 📌 Project Overview
This project is based on the study of Git, an open-source version control system. It includes both theoretical analysis and practical implementation using shell scripting.

---

## 💻 Scripts Included

### Script 1 — System Information
Displays system details like user, OS, kernel, and date.

### Script 2 — Package Inspector
Checks if Git is installed and shows version details.

### Script 3 — Directory Audit
Analyzes directories and shows their contents and permissions.

### Script 4 — File Analyzer
Counts occurrences of a keyword in a file.

### Script 5 — Manifest Generator
Creates a personalized open-source manifesto using user input.
